1. Flask-SQLAlchemy packages adds SQLAlchemy, along with some tie-ins to Flask, allowing programs to access databases
2. flask-marshmallow packages adds the Flask parts of Marshmallow, which lets programs convert Python objects to and from serializable structures.
3. marshmallow-sqlalchemy packages adds some Marshmallow hooks into SQLAlchemy to allow programs to serialize and deserialize Python objects generated by SQLAlchemy.
4. marshmallow packages adds the bulk of the Marshmallow functionality
